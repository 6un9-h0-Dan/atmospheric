Atmospheric Toolbox
===========

**Atmospheric Toolbox** is a set of tools for accessing, analyzing, and visualizing atmospheric data.
 
**[See the wiki](https://github.com/atmospheric/atmospheric/wiki)** for installation, tutorials, and more.
